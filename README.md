# 🛍️ ShoeZone – Online Shoe Shopping Web Application

ShoeZone is a user-friendly and interactive online shoe shopping web application developed using **Streamlit**. The application allows users to browse products, apply filters, manage cart and wishlist, and place orders efficiently.

---

## 🚀 Features

* 🔐 User Registration & Login
* 🔍 Product Search and Filtering
* 🛒 Add to Cart Functionality
* ❤️ Wishlist Management
* 💸 Coupon & Discount System
* 📦 Order Placement and Tracking
